# Smart Notes

Smart Notes is a privacy-first, offline desktop knowledge management app built with Electron.  
It stores notes as local Markdown files and supports modular AI-powered semantic search — without requiring the cloud.

## Features

- Local workspace (`.md` files as source of truth)
- Markdown editor (TipTap)
- Keyword search (Phase 1)
- Hybrid semantic search (Phase 2)
- Fully offline by default
- Optional OpenAI provider (user API key)

## Tech Stack

Electron · React · Vite · TipTap · MiniSearch · Ollama · SQLite (Vector DB) · OpenAI API(optional)
